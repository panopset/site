Build

You may build a platform installer from the source code. Run

  • ./build.sh on Linux,
  • build.cmd on Windows systems, or
  • ./buildmac.sh on Macintosh systems,

That will put an installer in the src project target directory.

It will also copy panopset.jar into your home directory. This is an all-in-one jar, that you may use with your JRE, should you not wish to use an installer.
